HTML

하이퍼 텍스트 마크업 언어(Hyper Text Markup Language, HTML, 문화어: 초본문표식달기언어, 하이퍼본문표식달기언어)는 웹 페이지를 위한 지배적인 마크업 언어다. 또한, HTML은 제목, 단락, 목록 등과 같은 본문을 위한 구조적 의미를 나타내는 것뿐만 아니라 링크, 인용과 그 밖의 항목으로 구조적 문서를 만들 수 있는 방법을 제공한다. 그리고 이미지와 객체를 내장하고 대화형 양식을 생성하는 데 사용될 수 있다. HTML은 웹 페이지 콘텐츠 안의 꺾쇠 괄호에 둘러싸인 "태그"로 되어있는 HTML 요소 형태로 작성한다. HTML은 웹 브라우저와 같은 HTML 처리 장치의 행동에 영향을 주는 자바스크립트와 본문과 그 밖의 항목의 외관과 배치를 정의하는 CSS 같은 스크립트를 포함하거나 불러올 수 있다.

HTML에 대한 문법

  • title태그 / html제목

  • head태그 / 머리부분

  • body태그 / 핵심html 코드 부분

  • ol태그 / 1. 2. 3. 등 목록

  • ul태그 / 기호를 쓰는 목록

  • h1(2,3,4)태그 / 문자의 제목 소제목 부분

  • strong태그 / 특정 문자를 굵게 하여 강조

  • u태그 / 특정 문자 밑줄 표시

  • p태그 / 단락

  • br태그 / 줄바꿈

  • img태그 / 이미지 추가

  • a태그 / url주소 태그 ====code형태====> a href="https://naver.com" target="_blank"> naver.com

    • href="( url 주소 )"
    • target="_blank"

      • =========css=========

    • class="saw" style / .saw

    • id="active" style / #active %한번만 사용%

  • from태그 / 전송모듈 ====code형태====> from action="( backend server url )" method="post"

    ====url형태====> http://( backend server url )/?id=gojomog&address=seoul

    • metode="post"
    • metode="get"
    • autocomplte="(on / off)"

    • input / 사용자 입력 ====code형태====> input type="text" name="id" value="아이디 autofocus required"

      • type="reset"

      • type="date" / name="datev"
      • type="month" / name="monthv"
      • type="week" / name="weekv"

  • footer태그 / 웹페이지의 하단에 추가 정보제공

  • nav태그 / 웹페이지의 네비게이션에 해당하는 정보

  • article태그 / 웹페이지의 본문에 해당하는 정보
